This prototype project enables data architects to view an ontology's datums and more complex data entities as HTML form elements, including term preferred name, definition, synonyms, database cross-references, categorical selection lists, and numeric data entry and validation fields.  Specifications for these elements can be generated for reuse by software developers; logged-in users can save selected elements in packages that can be updated and downloaded later in JSON or Excel format; and finally, a form generator script is provided.
